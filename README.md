# Army Defense AI

Automated surveillance and threat detection system using computer vision for perimeter security monitoring.

## Features

- Real-time object detection for intruder identification
- Motion-triggered recording and alerting
- Multi-camera feed aggregation
- Threat classification and severity scoring

## Tech Stack

Python, OpenCV, YOLO, TensorFlow

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Army-Defense-AI.git
cd Army-Defense-AI
pip install -r requirements.txt
```

### Usage

```bash
python detect.py --source camera_feed
```
